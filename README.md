# Retail-Data-Analytics
End-to-end retail data analytics project using Excel, SQL, Python, and Power BI.

## Project Overview
This project analyzes retail sales data to identify sales trends, seasonal patterns,
store performance, and the impact of holidays on sales.

## Tools & Technologies
- Excel (Data Cleaning)
- SQL (Data Validation & Analysis)
- Python (EDA & Visualization)
- Power BI (Dashboard & DAX Measures)

## Key Analysis
- Weekly and monthly sales trends
- Store-wise and type-wise performance
- Holiday vs Non-Holiday sales comparison
- KPI metrics using Power BI

## Outcome
The project provides actionable insights to understand sales behavior and
support data-driven business decisions.
